# ✈️ Airline Reservation System

## 📌 Overview
The **Airline Reservation System** is a full-stack web application built using **Java, Spring Boot, Hibernate, and MySql**.  
It enables users to search for flights, book tickets, manage reservations, and perform cancellations.  
The system also provides **secure login authentication**, **session management**, and **SMTP-based email verification** for registration and password recovery.

---

## 🚀 Features
- 🔍 **Flight Search & Availability** – View available flights based on criteria.  
- 🎫 **Ticket Booking & Cancellation** – Book and cancel tickets with transactional integrity.  
- 🔑 **User Authentication** – Secure login & logout with session management.  
- 📧 **Email Service** – SMTP-based email verification and password reset.  
- 🛡 **Data Security** – Consistency maintained using ACID properties and transaction management.  
- 🎨 **Responsive UI** – Designed with **HTML, CSS, Bootstrap, Thymeleaf**.  

---

## 🛠 Tech Stack  
**Backend**  
- Java  
- Spring Boot  
- Hibernate (ORM)  
- MySQL  

**Frontend**  
- HTML, CSS, Bootstrap  
- Thymeleaf (template engine)  

**Other**  
- SMTP (JavaMailSender for email service)
